## Use as atemplate

1. npm i
2. in package.json change the following

   ```
   "homepage": "https://<your-username>.github.io/<repository-name>/",

   ```

3. Do some changes
4. use Git or GitHubDesktop to save -> .add + commit (push origin)
5. npm run dev
6. npm run build
7. npm run preview
8. npm run deploy
